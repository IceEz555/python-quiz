ข้อที่ 1: การรับค่าและแสดงผลข้อความ

จงเขียนโปรแกรมรับค่า "ชื่อ" และ "นามสกุล" จากนั้นแสดงผลโดยนำมาต่อกันเป็นประโยค
ตัวอย่างผลลัพธ์ (Output Example)
enter your name: Jhon enter your surname: Doe Fullname is Jhon Doe
เฉลยโค้ด:
name = input("enter your name: ") surname = input("enter your surname: ") print("Fullname is " + name + " " + surname)
Python String

ข้อที่ 2: การเข้าถึงข้อมูลใน String

กำหนดให้ a = "Apple" จงเขียนคำสั่งเพื่อแสดงผล:
1. ตัวแรก
2. ตัวสุดท้าย
3. ตัดคำเอาเฉพาะ "le"
ตัวอย่างผลลัพธ์ (Output Example)
A e le
เฉลยโค้ด:
a = "Apple" print(a[0]) print(a[-1]) print(a[3:])
String Formatting

ข้อที่ 3: การจัดรูปแบบการแสดงผล (%)

กำหนด: name="Apivit", id="65502100109-5", grade=3.53
จงแสดงผลชื่อคู่กับรหัส และชื่อคู่กับเกรด (ทศนิยม 2 ตำแหน่ง)
ตัวอย่างผลลัพธ์ (Output Example)
Hi, Apivit ID is 65502100109-5 Hi, Apivit grade is 3.53
เฉลยโค้ด:
name = "Apivit" id = "65502100109-5" grade = 3.530 print("Hi, %s ID is %s" % (name, id)) print("Hi, %s grade is %.2f" % (name, grade))

StockCode Quantity total

เฉลยโค้ด:
pd.pivot_table(df, values=["total", "Quantity"], index=["StockCode"], aggfunc="mean")

ข้อที่ 5: Pivot Table แบบมีเงื่อนไข

กรองเฉพาะยอด total >= 1000 แล้วหา ผลรวม (sum) แยกตาม Country และ CustomerID
ตัวอย่างผลลัพธ์ (ตารางจำลอง)
Country CustomerID total
Australia12386.010799
12388.061747
.........
United Kingdom18278.02020
18287.037331
เฉลยโค้ด:
pd.pivot_table(df[df['total'] >= 1000], index=["Country", "CustomerID"], values="total", aggfunc="sum")

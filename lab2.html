ข้อที่ 3: เปรียบเทียบตัวเลข

รับตัวเลขจำนวนเต็ม 2 จำนวน เพื่อตรวจสอบเงื่อนไข:
1. ถ้า a > b ให้แสดงข้อความว่า "a มีค่ามากกว่า b"
2. ถ้าไม่ใช่ให้แสดงข้อความว่า "a มีค่าน้อยกว่า b"

Enter first number: 2 Enter second number: 6 a มีค่าน้อยกว่า b
a = int(input("Enter first number: ")) b = int(input("Enter second number: ")) if a > b: print("a มีค่ามากกว่า b") else: print("a มีค่าน้อยกว่า b")
Logic & Grading

ข้อที่ 4: โปรแกรมตัดเกรดนักเรียน

รับค่าคะแนน 3 ส่วน (เก็บ, กลางภาค, ปลายภาค) แล้วรวมคะแนนเพื่อตัดเกรดตามเกณฑ์

Enter Collect score: 20 Enter Midterm score: 6 Enter Final score: 15 Total score = 41.0 Grade = ไม่ผ่าน
collect = float(input("Enter Collect score: ")) midterm = float(input("Enter Midterm score: ")) final = float(input("Enter Final score: ")) total = collect + midterm + final print("Total score =", total) if total >= 80: print("Grade = A") elif total >= 75: print("Grade = B+") elif total >= 70: print("Grade = B") elif total >= 65: print("Grade = C+") elif total >= 60: print("Grade = C") elif total >= 55: print("Grade = D+") elif total >= 50: print("Grade = D") else: print("Grade = ไม่ผ่าน")
While Loop

ข้อที่ 5: การใช้ While Loop หาผลรวม

ใช้คำสั่ง while loop เพื่อหาผลรวมของตัวเลขจำนวนเต็มตั้งแต่ 0 ถึง 10

number = 0 number = 1 ... number = 10 Sum total = 55
a = 0 total = 0 while a <= 10: print(f"number = {a}") total += a a += 1 print("Sum total = ", total)
For Loop (List)

ข้อที่ 6: การใช้ For loop ใน List

กำหนดให้ List = [1, 10, 20, 30, 40, 50] จงหาผลรวมของตัวเลขทั้งหมด

Sum total = 151
data_list = [1, 10, 20, 30, 40, 50] total = 0 for i in data_list: total += i print("Sum total = ", total)
